# Tourism Marketing Data Analysis & AI Integration

## Project Overview
This project focuses on transforming raw Indian tourism data into actionable marketing insights. By combining traditional data analysis with AI-assisted tools, I analyzed 1,000 marketing campaigns to identify how "Tone of Voice" aligns with different "Target Audiences."

## Key Deliverables
**Cleaned Dataset**: Processed version of the Indian Travel Dataset with standardized categories.
**Prompt Library:** Documentation of the prompt engineering techniques used to generate insights.

## Tools Used
**Data Handling:** CSV/Excel
**AI Integration:** Microsoft Copilot / Gemini (for insight generation and content summarization)

## Main Insight
The analysis revealed a highly segmented marketing strategy where communication tones (Romantic, Peaceful, Exciting) are strictly mapped to specific traveler personas (Couples, Seniors, Families).
